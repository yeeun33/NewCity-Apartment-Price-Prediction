# 신도시 아파트 가격 예측
 
> 2기 신도시 개발 단계별 아파트 실거래가 데이터를 기반으로 머신러닝 모델을 활용하여 아파트 가격을 예측하는 데이터마이닝 프로젝트

 
## 프로젝트 개요
 
신도시 개발이 아파트 가격에 미치는 영향을 분석하고, 다양한 변수를 활용하여 아파트 거래금액을 예측합니다.  
국토교통부 실거래가 공개시스템 데이터를 기반으로 피처 엔지니어링부터 모델 학습까지 전 과정을 수행합니다.
 
---
 
## 분석 대상 신도시
 
| 신도시 | 위치 |
|--------|------|
| 판교 | 경기도 성남시 분당구 백현동·판교동 |
| 광교 | 경기도 수원시 영통구 이의동 |
| 송도 | 인천광역시 연수구 송도동 |
| 청라 | 인천광역시 서구 청라동 |
| 운정 | 경기도 파주시 와동동·목동동·야당동 |

```
|   └── 세대수.ipynb                 # 단지별_세대수, 도시별_세대수 변수 추가
|   └── 가장가까운IC와의거리.ipynb      # 단지 기준으로 가장 가까운 IC와의 거리 변수 추가
|   └── 가장가까운지하철역까지의거리.ipynb  # 단지 기준으로 가장 가까운 지하철역까지의 거리 변수 추가
├── models/
|   ├── Lasso/

│   |   └── lasso_whatif_geomdan.ipynb  # 파밀리아 엘리프 시나리오 예측(What-If 분석)
|   ├── Ridge/
|   |   └── ridge_validation.ipynb      # Ridge 검증 코드
|   |   └── ridge_verify_geumgang.ipynb # 금강펜테리움 센트럴파크 시계열 검증 코드
|   |   └── ridge_whatif_geomdan.ipynb  # 파밀리아 엘리프 시나리오 예측(What-If 분석)
└── README.md
```

## 피처 엔지니어링
 
> 팀원별로 변수를 분담하여 원본 데이터에 추가하였습니다.
 
| 변수명 | 설명 | 수집 방법 | 담당 |
|--------|------|-----------|------|
| `CPI` | 소비자물가지수 (2020년=100 기준) | 통계청 소비자물가지수 공식 누리집 | 박소정 |
| `발표후경과년수` | 신도시 지구지정 고시 후 경과 연수 | 국가기록원 및 지자체 도시공사 사업개요 자료 기반, 계약연도와의 차이로 계산 | 박소정 |
| `서울도심거리` | 광화문(서울 도심 기준)까지의 거리 (km) | 카카오 로컬 API + Haversine 공식으로 계산 | 김예은 |
| `단지별_세대수` | 아파트 단지의 총 세대수 | K-apt 공동주택 단지 기본정보 CSV + 건축물대장 표제부 단지명 유사도 매칭 | 김예은 |
| `도시별_세대수` | 신도시 전체 계획 세대수 | K-apt 공동주택 단지 기본정보 CSV + 건축물대장 표제부 단지명 유사도 매칭 | 김예은 |
| `가장가까운IC와의거리` | 아파트 단지에서 가장 가까운 고속도로 IC까지의 거리 (km) | 카카오 로컬 API 사용하여 직선거리 반환 | 문인선 |
| `가장가까운지하철역까지의거리` | 아파트 단지에서 가장 가까운 지하철역까지의 거리 (km) | 카카오 로컬 API 사용하여 직선거리 반환 | 문인선 |

## Lasso 모델 결과
 
**목표 변수**: `㎡당가격 (전용면적당 거래금액)`
 
### 모델 학습 과정
1. 목표 변수를 거래금액에서 m2당가격으로 변경하여 도시 간 가격 수준 비교 가능하도록 설계
2. z-score 기반 이상치 제거 (|z| > 2, 95% 신뢰구간 밖 데이터 제거)
3. alpha = [0.001, 0.01, 0.1, 1, 10, 100] 범위에서 최적 alpha 탐색
4. 최적 alpha = 0.001로 최종 모델 학습

### 성능 결과

#### 1. 모델 검증(train_test, 8:2 랜덤 분할)
| 지표 | 훈련 세트 | 테스트 세트 |
|------|-----------|-----------|
| R² | 0.7461 | 0.7424 |

#### 2. 실거래 검증(금강펜테리움센트럴파크, 2026년 38건 평균)
> 2026년 실거래 데이터가 있는 신검단중앙역 금강펜테리움센트럴파크(38건 평균)와 예측값을 비교하여 모델 신뢰성을 검증함

| 구분 | 금액 |
|------|------|
| 실제 거래금액 | 52,093만원 (5.2억) |
| 예측 거래금액 | 54,691만원 (5.5억) |
| 오차율 | 5.0% |

#### What-if 시나리오 분석
> 인천 검단 2027년 완공예정인 파밀리에 엘리프(AA32블록, 84㎡ 중층 기준)에 대해 시나리오별 예측을 수행함

| 지표 | 훈련 세트 | 테스트 세트 |
|------|-----------|-----------|
| R² | 0.8221 | 0.7291 |

| 시나리오 | 예측 금액 | 
|---------|------------|
| 현재 | 56,433만원 (5.6억) |
| 발표 후 25년 | 62,183만원 (6.2억) |
| 지하철 추가 | 62,664만원 (6.3억) | 
| 복합 시나리오 | 68,414만원 (6.8억) |
